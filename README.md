# Employee Management System

MERN-style employee management app with a React frontend and an Express/Apollo/MongoDB backend.

## Setup

Backend:

```powershell
cd ems-back
npm install
Copy-Item .env.example .env
npm start
```

Frontend:

```powershell
cd ems-front
npm install
npm start
```

Update `ems-back/.env` with your real MongoDB connection string before starting the backend.

## Build

```powershell
cd ems-front
npm run build
```
